PCB Layout Review Service

Signal integrity, return paths, EMI, thermal behaviour, and DFM - reviewed by engineers who've debugged board failures at the production floor.

What DRC can't catch

DRC tells you if your layout is manufacturable. It says nothing about whether it will work. Signal integrity, EMI performance, thermal reliability, and power delivery are all determined by layout decisions that no automated tool checks. That is what a PCB layout review is for.

Return paths and ground plane analysis

Every signal current has a return current - and at high frequencies, it wants to flow directly beneath the signal trace. When the return path is broken by a plane slot, a split, or a missing stitching via at a layer change, that return current detours into a longer loop - creating a loop antenna. We analyse return current paths for all high-frequency and high-current nets, and identify every discontinuity that affects EMI or signal integrity.

Signal integrity and high-speed routing

For designs with USB, Ethernet, DDR, HDMI, or high-speed SPI, we review controlled-impedance trace geometry for target compliance, differential pair routing for spacing and length matching, layer transitions for impedance and return path impact, and via stub length where backdrilling is a consideration.

Antenna matching and RF layout

RF layout tolerances are tight. Transmission line length to the antenna feed, matching network component placement relative to the RF port, ground plane geometry beneath the antenna, and clearance from interference sources all affect performance significantly. We review RF layouts for compliance with IC and module integration guidelines, and flag cases where the physical layout deviates from the reference design in ways that matter.

EMI and EMC risk assessment

Most EMI problems originate in layout decisions - not schematic errors. Unfiltered clock signals adjacent to IO lines, switching converter output traces without filtering, common-mode chokes placed after the cable connector, and inadequate stitching capacitors between split power planes are all sources we systematically look for. Our team has experience with CE (CISPR 32) and FCC Part 15 pre-compliance testing. We review with knowledge of what the test chamber will measure.

PCB stack-up and technology

The stack-up choice has major implications for signal integrity and EMI. We verify that signal layers have adjacent reference planes, that the dielectric and trace geometry match impedance targets, and that the overall stack-up suits the design's frequency content. Where stack-up is not finalised, we provide specific recommendations.

Thermal management

Thermal failures are intermittent and worsen with age. We verify exposed-pad implementation for all packages requiring direct thermal connection, check via array sizing and placement for heat transfer, assess copper pour area relative to power dissipation, and verify that the thermal design is consistent with the expected operating environment - not just lab conditions.

DFM - Design for Manufacturability

DFM errors range from minor (cost penalty) to critical (non-assembable boards). We check solder mask expansion ratios, courtyard clearances, paste mask aperture sizing for fine-pitch packages, fiducial presence and placement, solderjoint accessibility for rework, and board edge clearances for routing and v-scoring. For designs targeting specific EMS partners, we can apply their DFM rules to the review.

Example findings

Typical issues we identify

GND plane slot under high-speed trace Critical

A slot in the ground plane directly beneath a 50MHz UART trace forces the return current into a 14mm loop - creating a loop antenna. This is a common root cause of CE radiated emission failures.

→ Fix: Re-route trace to avoid slot, or fill slot and add stitching vias on either side.
Decoupling cap 8.2mm from IC power pin Critical

The 100nF cap for the core power rail of U3 is 8.2mm from the VDD pin. At 100MHz, trace inductance dominates - the capacitor provides minimal decoupling where it is needed most.

→ Fix: Move within 0.5mm of VDD pin. Use via-in-pad if space is constrained.
Insufficient thermal vias - MOSFET pad Advisory

Q2 has 4 thermal vias. At peak load of 8A, junction temperature exceeds the rated maximum by ~15°C under worst-case ambient. Minimum recommended count is 12.

→ Fix: Increase to ≥12 vias at 0.3mm drill. Extend copper pour on inner layers.
Solder paste aperture too large - QFN-28 Advisory

The paste aperture for U6 exposed pad is at 100% coverage. Standard guidelines recommend 60–75% with a segmented pattern to prevent solder voiding and tombstoning during reflow.

→ Fix: Segment aperture into 3×3 grid at 65% total coverage.
